Un1ts

Un1ts is a comprehensive unit converter for the Mac. The app is built with Swift and SwiftUI to support the latest features of macOS. Just enter a value in a field then press the Return key to automatically convert to all units in a category.

Categories for unit conversion include:

  • Acceleration
  • Angle
  • Area
  • Data
  • Force
  • Length
  • Mass
  • Pressure
  • Radioactivity
  • Speed
  • Temperature
  • Volume
  • Weight
