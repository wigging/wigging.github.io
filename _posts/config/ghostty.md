---
title: Ghostty Configuration
date: April 28, 2025
---

Ghostty is the best terminal application for the Mac. See below for my configuration settings. More
information about Ghostty is available at [ghostty.org](https://ghostty.org).

```bash
# Configuration file for Ghostty
# On macOS this file is located at ~/.config/ghostty/config
# See https://ghostty.org

adjust-cursor-thickness = 2
adjust-cell-height = 3

background = #2A2F3A

font-family = Menlo
font-size = 13
font-style = Regular
font-style-bold = Regular
font-style-italic = Regular
font-style-bold-italic = Regular
font-thicken

macos-titlebar-style = native

theme = xcodedarkhc

window-colorspace = display-p3
window-height = 46
window-width = 140
window-theme = system
```
